# name-pascal-class

> Use PascalCase for class names.

## Why
PEP 8 mandates PascalCase for classes. Distinguishing classes from functions visually helps readability.

## Bad
```python
class user_repository:
    ...

class data_pipeline:
    ...
```

## Good
```python
class UserRepository:
    ...

class DataPipeline:
    ...
```
